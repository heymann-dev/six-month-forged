# Deploy Project Northstar v30 to GitHub Pages

1. In the current live Northstar app, use **Data → Export all JSON + photos**.
2. Download and unzip `Project_Northstar_v30_GitHub_Ready.zip`.
3. Open the `heymann-dev/six-month-forged` repository.
4. Choose **Add file → Upload files**.
5. Upload the contents of the extracted folder—not an enclosing `northstar_v30` folder.
6. Confirm `index.html`, `manifest.json`, and `service-worker.js` appear at the repository root and the complete `assets` folder is included.
7. Commit with `Update Project Northstar to v30`.
8. Wait for the GitHub Pages deployment workflow to complete successfully.
9. Open `https://heymann-dev.github.io/six-month-forged/?v=30`.
10. Verify History, Goals, Recovery, Mountains, Cash, Photos, Reviews, Tasks, and sealed rewards.
11. Export a new full v30 backup.

Do not clear Safari website data. Removing and re-adding the Home Screen shortcut is safe after the live v30 page has loaded, but clearing website data can remove locally stored records.
